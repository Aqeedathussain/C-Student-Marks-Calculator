## Description
This repository contains a simple C++ program that calculates the **aggregate** and **average** marks of a student for five subjects.  
It demonstrates:
- Arrays to store marks
- Looping for input and sum calculation
- Basic arithmetic operations
- Output formatting using `cout`

This program is ideal for beginners learning C++ basics.

---

## Features
- Accepts input marks for 5 subjects.
- Calculates the **aggregate** marks.
- Calculates the **average** marks.
- Displays results in a clean format.
